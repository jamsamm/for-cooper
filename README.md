## How to Customize

### 1. Edit the Letters
Each letter file in the `letters/` folder has placeholder text. Open any letter file and replace the content with your own heartfelt message.

### 2. Change the Background Image
In `styles.css`, find the `.background` class and replace the image URL with your own.

### 3. Customize Colors
Look for `#ff1493` (hot pink) and `#d63384` (deep pink) throughout `styles.css` and replace with your preferred colors.

### 4. Personalize Text
Edit the main message, signature, and Polaroid caption in `index.html`.

For detailed customization instructions, see [CUSTOMIZATION.md](CUSTOMIZATION.md).

## Troubleshooting

**Photos not showing?**
- Check the image path is correct
- Use PNG or JPG formats
- Ensure file is in the correct folder

**Colors not updating?**
- Clear your browser cache (Ctrl+F5 or Cmd+Shift+R)
- Make sure you're editing the CSS files

**Letters not linking?**
- Verify the file name matches exactly
- Check files are in the `letters/` folder
- Ensure file extensions are `.html`
